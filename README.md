
# Notes App  
  
A simple notes / todo app based on Flutter built as part of Raro Academy Training Course.  
  
## About  
  
This project intent to show user's notes.  
  
## Core functions  

 - Add item ✅
 - Swipe to edit item ⏳
 - Swipe to remove item ⏳
 - Change item icon ⏳
